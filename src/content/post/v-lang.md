[The V Programming Language](https://vlang.io/)
[docs](https://vlang.io/docs)

- Go-like syntax
- compiles to human readable C
- C compatibility
- manual memory allocation
- autofree at compile time to reduce GC at runtime
- generics
- optionals (explicitly null pointers)
