## eopkg

[solus-project/package-management](https://github.com/solus-project/package-management)

The packages are to be installed with `sudo eopkg install -y`.

```
git vim
ghex atom
xorg-driver-video-intel libva-intel-driver
ffmpeg smplayer
ibus ibus-cangjie
steam
```

```sh
# Update
sudo eopkg up
```
